# CURL

This Repository will keep track of simple tools found within Curl.

## Requirements

- Git
- Curl tool installed

## CURL Commands

|                commands                      |                   Explanation                              |
|----------------------------------------------|------------------------------------------------------------|
| curl --help                                  |    Shows the curl help list                                |
| curl <*Any URL*>                             |    This will return the HTML of the URL choosen.           |
